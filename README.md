# 🖐️ WebXR Hand Physics Laboratory

A high-performance, browser-based XR experience where you can interact with 3D primitives using **real-time hand tracking** and physics. Built with A-Frame and the WebXR Hand Input API.

## 🚀 Live Demo
[**Click here to view the experience**](https://your-username.github.io/your-repo-name/)
*(Note: Requires a WebXR-compatible headset and browser like Meta Quest, Apple Vision Pro, or Chrome on Android)*

---

## ✨ Features
* **Full Hand Tracking:** Uses the WebXR Hand Input API to render actual 3D mesh hands mapped to your real movements.
* **Physical Interactions:** Push, poke, and flick objects. Uses the `aframe-physics-system` (Cannon.js) for real-time collision detection.
* **Geometric Primitives:** Interact with cubes, spheres, and 3-sided pyramids (triangles).
* **Immersive Environment:** A procedural forest environment to provide depth and spatial context.

## 🛠️ Tech Stack
* [**A-Frame**](https://aframe.io/) - Web framework for building VR experiences.
* [**WebXR API**](https://immersiveweb.dev/) - Device communication.
* [**C-Frame Physics**](https://github.com/c-frame/aframe-physics-system) - Physics engine for A-Frame.

---

## 🎮 How to Use
1.  **Open the Link:** Use your headset's browser (e.g., Quest Browser, Wolvic).
2.  **Enter VR:** Click the **"VR"** button at the bottom right of the screen.
3.  **Track Hands:** If using a headset like Quest, put your controllers down or enable hand tracking in your system settings.
4.  **Interact:** Simply move your hands to push the objects off the platform!
